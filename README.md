# Matrix
## Info
This repository implements the matrix determinant search algorithm  
Input: the matrix size.   
Output: matrix determinant.   
## Build a program
`cmake -S ./ -B build`  
`cd build`  
`make`
## Build unit tests
`cd tests`   
`make` to make tests  
`make huge` to make huge test 

## Build unit tests
`cd tests`   
`cmake -S ./ -B build`

`make`
